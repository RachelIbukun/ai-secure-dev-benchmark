# Initial Reading List

This reading list supports the research project: **Evaluating Security Risks in AI-Assisted Software Development Pipelines**.

## Key Research Themes

* AI-assisted software development
* Security risks of AI-generated code
* Software supply-chain security
* Developer productivity and security trade-offs
* Governance of AI-assisted engineering workflows
* Frontier model evaluation
* Secure deployment of advanced AI systems

## Questions Guiding the Review

1. What is currently known about the productivity benefits of AI coding assistants?

2. What security vulnerabilities are commonly introduced by AI-generated code?

3. How do AI-assisted workflows affect developer judgement and review behaviour?

4. Are existing security tools sufficient to detect AI-generated weaknesses?

5. What governance controls should organisations implement before deploying AI-assisted development tools at scale?

## Working Hypothesis

AI-assisted software development can improve short-term productivity, but without appropriate review, monitoring, and governance, it may introduce hidden security risks and long-term technical debt.
